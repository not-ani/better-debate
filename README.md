# Better Debate

A set of software built for debaters by debaters.

1. Better word

Better word is a fully custom replacement for verbatim and microsft word. It is built for the peropuses of creating cards for debate and debating.

# electrobun-monorepo

Monorepo using Bun workspaces with:

- `apps/desktop` (Electrobun desktop app)
- `packages/core` (Rust cdylib + Bun FFI)
- `packages/ui-svelte` (Svelte + Vite view)
- `packages/ui-solid` (Solid + Vite view)

Commands:

```bash
bun install
bun run dev
bun run build
bun run clean
```
